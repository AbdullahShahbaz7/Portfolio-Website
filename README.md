# My Portfolio - React + Framer Motion

A modern animated portfolio built with React and Framer Motion.

## ✨ Features
- **Hero Section** – Animated intro with resume download & GitHub link  
- **Services** – Interactive cards with hover effects  
- **Portfolio** – Project showcase with live demos & tech tags  
- **Contact** – Simple form and social links  
- **Responsive** – Navbar with mobile sidebar, smooth scrolling  

## 🛠️ Tech Stack
React • Framer Motion • SCSS • Vite • React Icons

## 🚀 Quick Start
```bash
git clone <repo>
npm install
npm run dev
```

## 🔧 Customize
- Update text in component files  
- Replace images in `assets/`  
- Add projects in `portfolio` component  
- Edit styles in `.scss` files
